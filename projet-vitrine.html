Site Vitrine "Gridiron"

Projet réalisé en première année de BTS SIO, consistant à créer une landing page responsive pour une boutique fictive de maillots et accessoires de football américain. L'objectif était de mettre en pratique les bases du développement web front-end.

HTML CSS Bootstrap Responsive

Contexte

Ce projet s'inscrivait dans le module de développement web. Il fallait concevoir une page d'accueil statique mais attrayante, en utilisant le framework Bootstrap pour assurer la responsivité sur différents formats d'écrans (mobile, tablette, desktop).

Technologies

  • HTML5 pour la structure sémantique.
  • CSS3 pour le style personnalisé et les animations.
  • Bootstrap 5 pour le système de grille et les composants.
  • JavaScript (très léger) pour des interactions simples.

Période & Groupe

Réalisé sur une période de 2 semaines en Octobre 2024. Ce projet a été mené en autonomie, permettant de maîtriser chaque étape du processus de création.

Galerie d'images

Voici quelques captures d'écran du projet, montrant la version desktop et mobile.

Conclusion & Apprentissages

Ce projet a été une excellente introduction à la conception de sites web modernes. Il m'a permis de consolider mes compétences en HTML/CSS et de découvrir la puissance de Bootstrap pour créer des designs responsives rapidement. La principale difficulté a été d'assurer une compatibilité visuelle parfaite sur tous les navigateurs, un défi qui m'a beaucoup appris sur le débogage CSS.
