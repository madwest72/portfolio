projet personnel

Letterbook

Application web de type CRUD inspirée de Letterboxd, mais pour les livres. Le projet intègre l'API Google Books pour récupérer les informations sur les livres et propose des carrousels par genre ainsi qu'un fil d'activité.

PHPSQLGoogle APIBootstrap

Contexte

Étant un grand lecteur, j'ai voulu créer un outil pour suivre mes lectures et partager mes avis. Ce projet personnel est né de l'envie de combiner ma passion pour les livres et pour le développement web.

Technologies & Outils

  • PHP et SQL pour le back-end et la base de données.
  • API Google Books pour la recherche et l'import de données.
  • cURL en PHP pour interroger l'API.
  • Bootstrap pour une interface responsive.

Projet personnel développé sur mon temps libre.
