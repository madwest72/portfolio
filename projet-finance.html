projet école

Gestion Finance

Application web permettant de gérer ses finances personnelles. Le projet inclut une base de données pour le suivi des transactions et un back-office pour les opérations CRUD (Create, Read, Update, Delete).

PHPSQLHTMLCSS

Contexte

Ce projet de première année avait pour but de nous familiariser avec la création d'applications web dynamiques connectées à une base de données. L'objectif était de construire un outil simple mais complet pour la gestion financière personnelle.

Technologies & Outils

  • PHP pour toute la logique serveur.
  • SQL (via MariaDB/MySQL) pour la persistance des données.
  • HTML et CSS pour la structure et la présentation.
  • Un back-office pour la gestion des données.

Période & Groupe

Projet réalisé en autonomie en première année de BTS.

Conclusion & Apprentissages

Ce projet a été fondamental pour mettre en pratique les opérations CRUD, qui sont au cœur de nombreuses applications web. J'ai pu construire une application back-end fonctionnelle de bout en bout, de la base de données à l'interface utilisateur.
