Helpdesk GLPI + n8n

Projet personnel de mise en place d'un système de helpdesk complet. Il utilise GLPI pour la gestion des tickets et n8n pour automatiser leur création et leur routage à partir d'une boîte mail. L'ensemble est déployé via Docker.

DockerGLPIn8nGmail API

Contexte

L'idée de ce projet était de simuler un environnement de support IT en utilisant des outils open-source professionnels. Je voulais automatiser le flux de création de tickets pour le rendre plus efficace, comme en entreprise.

Technologies & Outils

  • Docker et Docker Compose pour l'orchestration.
  • GLPI pour le système de ticketing (ITSM).
  • n8n pour l'automatisation des workflows.
  • API Gmail pour déclencher les workflows à la réception d'un email.

Période & Groupe

Projet personnel réalisé en autonomie, en cours d'amélioration continue.
