projet école

DumbFit

Plateforme de suivi de santé qui permet de monitorer les objectifs de bien-être avec une logique de tranches de performance.

PHPSQLJSMVCBootstrap

Contexte

Dans le cadre d'un projet de 1ère année, il fallait développer une application web permettant à des utilisateurs de suivre leurs performances de santé et de bien-être, avec une logique métier spécifique basée sur des "tranches" de performance.

Technologies & Outils

  • PHP pour la logique back-end.
  • SQL pour la gestion des données utilisateur.
  • JavaScript pour l'interactivité côté client.
  • Architecture MVC pour structurer le code.
  • Bootstrap pour l'interface responsive.

Période & Groupe

Projet réalisé en autonomie en première année de BTS.

Conclusion & Apprentissages

Ce projet a été l'occasion d'implémenter une logique métier plus complexe et de structurer le code en suivant le patron d'architecture MVC. La gestion des différentes "tranches" de performance a été un défi intéressant à relever.
